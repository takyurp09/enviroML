# Changelog

All notable changes to enviroML are documented here.

## [1.1.0] — 2026-07-21

### Added

- Five-fold station-held-out regression validation
- Post-2016 temporal holdout and 1,000-resample RMSE confidence interval
- Seasonal residual analysis
- Voting regression and classification ensembles
- Persistence and climatology climate baselines
- Expanding-window climate validation
- Delaware Estuary monitoring-network map
- Bangladesh grid-cell error map
- Model and data cards

## [1.0.0] — 2026-07-21

### Added

- Unified Delaware Estuary and Bangladesh ERA5 case studies
- Regression, classification, clustering, interpretation, and climate workflows
- Held-out and chronological validation designs
- Cluster-selection and 200-refit stability analysis
- Six generated figures and seven machine-readable evidence tables
- Reproducibility, provenance, data-rights, and validation documentation
- Pinned Python and Conda environments
- Automated checks across Python 3.11–3.13
- Station-grouped and later-year validation with bootstrap uncertainty
- Voting ensembles and explicit climate baselines
- Seasonal error analysis and expanding-window climate validation
- Delaware Estuary station map and Bangladesh spatial error map
